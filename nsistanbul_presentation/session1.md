## Installation

### Vapor

```bash
$ brew tap vapor/tap
$ brew install vapor/tap/vapor
```

### Create Vapor Project

```bash
$ vapor new NSDeprem
$ cd NSDeprem
```

### Generate Xcode Project

```bash
$ vapor xcode -y
```
